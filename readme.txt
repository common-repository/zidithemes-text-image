== Description ==

Zidithemes Text Image is designed to improve the layout of client text images, team, profile and works with the [Gutenberg editor](https://wordpress.org/plugins/gutenberg/ "Gutenberg Plugin"). This custom block is designed to display text images. It has options to add an image, title and description. The Zidithemes text image is a superb solution for creating text images.

== Installation ==

Note: This plugin requires the Gutenberg plugin active.

1. Upload 'Zidithemes text image' to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Within a post or page, select the '+' symbol to add a new block.
4. From the 'Blocks' tab, select the 'zidithemes text image' block.
5. Add image, title, description.
6. Within the 'Zidithemes text image Color' section on the right, adjust the color.
6. Within the 'Zidithemes text image Settings' section on the right, adjust the text margin and image position.

== Frequently Asked Questions ==

= Q: Does this plugin require the Gutenberg plugin? =

A: Yes, the Gutenberg plugin must be installed and active in order to use this block.

= Q: Can I use the zidithemes text image Block with themes other than Zidithemes Themes? =

A: Yes, the block will work and transfer between themes.
